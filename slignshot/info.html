Planetary System Creator

Simulation of gravitational interactions between objects by using the law of universal gravitation. The simulation is presented graphically using a canvas element refreshed at a specified frequency(milliseconds), the refresh rate can be changed in the options. In order to create a new system, select "Create new system"
after selecting it the system creation window will be displayed, adding objects and displaying their information is done using the various buttons from the navigation menus.
All menus use the "getElementByID" and "innerHTML" functions, each button uses the on..() function and operations on style properties, each button also uses the "audio" element, hover sounds and button press sounds, in case of no sound, i.e. the buttons do not play audio elements, you need to uncheck the "Block audio auto play" option in your browser. The creation of an object is done after selecting and filling in its data and selecting the "Place" button.

The object's flight path is predicted in a separate thread using the "Web Workers" technology.

Detailed information about each object and options for removing the object can be double-displayed by clicking on the object on the canvas or selecting it from the right menu displaying all created objects.

Information about the selected object is displayed live in text form when you hover the mouse over the object.

In the event of a collision, the object with the smaller mass is destroyed.

As the mass of an object increases, its radius increases, and in the case of stars, their colors change based on the spectral classes of the stars.

When creating a planet, you can choose one of 7 types, including the ring type Opening and closing the menu is done using the "style.display" style property.
